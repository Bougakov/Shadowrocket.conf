# Shadowrocket.conf

Вычищенная вручную компиляция из разных списков для Shadowrocket под использование на территории РФ. На телефоне достаточно установить короткий конфиг вида:

```ini
[General]
update-url = https://github.com/Bougakov/Shadowrocket.conf/raw/main/RU.conf
```

Импортируте его в телефон, затем выделите и нажмите Update в контекстном меню. Должен отобразиться размер около 430 килобайт. Для проверки используйте функцию _Test rule_: проверьте на доменах `2ip.ru`, `2ip.io`, `2ip.me` и `2ip.ua` - соединение к ним пойдёт по-разному.

Преимущества этого конфига:

- серверы разделяются на группы: трафик до Ютуба пойдёт через серверы в России, трафик до ChatGPT - через серверы, в названии которых есть "AI" или "Нейросети"
- трафик до серверов Apple идёт напрямую
- для доступа к Телеге Shadowrocket будет выбирать только те сервера, которые не забанены в сервисе
- "народные" списки хостов по возможности вычитаны, из них убраны глупости и повторы
- в качестве основного DNS используется Гугл и Cloudflare, при недоступности - Яндекс.ДНС

Если на домашнем роутере установлен [NFQWS2](https://github.com/nfqws/nfqws2-keenetic#nfqws2-keenetic) и разблокированы Ютуб и Инста, можно использовать альтернативный конфиг:

```ini
[General]
update-url = https://github.com/Bougakov/Shadowrocket.conf/raw/main/RU-noDPI.conf
```

В этом случае можно воспользоваться функцией Shadowrocket **"Scene"** и автоматически выбирать второй конфиг для домашней сети и первый - для всех остальных.
